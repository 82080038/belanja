# Daftar Kebutuhan Komputer

## Deskripsi
Daftar perlengkapan komputer yang masih kurang untuk beberapa unit komputer.

## Kategori Kebutuhan

### Hardware Komputer
- [ ] RAM tambahan (4GB DDR4)
- [ ] SSD 256GB untuk komputer #2
- [ ] VGA card entry level
- [ ] Power supply 450W
- [ ] CPU cooler
- [ ] Motherboard socket LGA 1151

### Aksesoris
- [ ] Mouse gaming (2 unit)
- [ ] Keyboard mechanical (3 unit)
- [ ] Monitor LED 24" (1 unit)
- [ ] Webcam HD
- [ ] Microphone
- [ ] Speaker aktif

### Kabel & Konektor
- [ ] HDMI cable (3 meter)
- [ ] USB extension cable
- [ ] Power strip 6 slot
- [ ] LAN cable 10 meter
- [ ] USB hub 4 port

### Software & Lisensi
- [ ] Windows 11 Pro license
- [ ] Microsoft Office 2021
- [ ] Antivirus license
- [ ] Adobe Photoshop license

## Status Komputer
- **Komputer #1**: Kurang RAM dan SSD
- **Komputer #2**: Butuh VGA card dan monitor
- **Komputer #3**: Perlu upgrade motherboard
- **Komputer #4**: Lengkap, hanya perlu software

## Prioritas Pembelian
1. Hardware critical (RAM, SSD, VGA)
2. Aksesoris penting (mouse, keyboard)
3. Kabel dan konektor
4. Software dan lisensi

- Total hardware: Rp 5.000.000
- Total aksesoris: Rp 2.500.000
- Total software: Rp 3.000.000
- **Grand total: Rp 10.500.000**
